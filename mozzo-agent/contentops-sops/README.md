## How to Update SOPs

1. Edit the relevant `.md` file
2. Add a changelog entry at the bottom of the file with date + what changed
3. Push to main → Mavi and team see it immediately
4. If it's a major change: post a summary in #ops-briefing

## Version Control

Every change is versioned by git. If you need to revert to an older version of any SOP,
check the git history for that file.
